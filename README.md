# Habithon

## Installation & Usage

### Installation

- Clone or download the repo.
- Open terminal and navigate to `fizzbuzz` folder.
- Run `npm install` to install dependencies.

### Usage

- Run `npm start` to launch server.
- Run `npm test` to launch test suite.
